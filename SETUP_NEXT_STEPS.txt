OPTION A: Easiest (30 seconds with service key)
─────────────────────────────────────────────
1. Find your Service Role Key:
   → See: FIND_SERVICE_KEY.md in project root

2. Add to .env file:
   SUPABASE_SERVICE_KEY=your_key_here

3. Run one command:
   npm run migrate:products

✅ Done! Tables created + data migrated


OPTION B: Manual (2 minutes without service key)
────────────────────────────────────────────
1. Go to Supabase dashboard:
   https://supabase.com/dashboard

2. Open SQL Editor → New Query

3. Copy & paste SQL from:
   scripts/migrate-to-variants.sql

4. Click "Run"

5. Back in terminal, run:
   npm run migrate:products

✅ Done! Tables created + data migrated
